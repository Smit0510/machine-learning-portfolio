# Machine Learning Portfolio: Classification & Regression

A collection of end-to-end Machine Learning projects focused on predictive modeling, feature engineering, and data analysis using classical ML algorithms.

## Projects Included

* **Loan Prediction:** Classification model for financial risk assessment.
* **Kidney Disease Prediction:** Predictive modeling for healthcare diagnostics.
* **Car Price Estimator:** Regression model for market value analysis.
* **Laptop Price Prediction:** Regression analysis for electronic device pricing.
* **Insurance Premium Prediction:** Predictive modeling for healthcare insurance costs.

## Tech Stack
* **Languages:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
* **Tools:** Jupyter Notebook

## Technical Notes
- **Notebooks:** All outputs have been cleared to ensure repository health and readability.
- **Model Files:** Serialized models (.pkl) are excluded from this repository; training logic and model evaluation are fully documented in the respective notebooks.
